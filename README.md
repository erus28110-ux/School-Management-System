# 🏫 School Management System

> نظام إدارة مدرسة متكامل

## 📋 الوصف
نظام سطح مكتب لإدارة المدارس يشمل تسجيل الطلاب،
الجداول الدراسية، النقاط والتقارير.

## ⚙️ التقنيات المستخدمة
- **Delphi 12** — الواجهة والمنطق البرمجي
- **Microsoft Access** — قاعدة البيانات

## ✨ المميزات
- تسجيل الطلاب وملفاتهم
- إدارة الأقسام والمواد
- إدخال النقاط وإصدار التقارير
- سجلات الأساتذة والموظفين
